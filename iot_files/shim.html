¿Qué es IoT? (Internet de las Cosas)

Dispositivos físicos (sensores, actuadores, PLCs, etc.) que recopilan y transmiten datos del mundo real, permitiendo monitorear y controlar objetos y entornos de forma remota.

¿Qué son los Gemelos Digitales?

Réplicas virtuales dinámicas de activos físicos, procesos o sistemas. Reflejan su estado en tiempo real o histórico, permitiendo análisis, simulación y optimización.

Este dashboard actúa como la capa de visualización y control, permitiendo representar estos gemelos de forma modular y personalizable.

Integración con el Mundo Físico

El sistema consume datos en tiempo real o diferido desde diversas fuentes IoT:

  • Dispositivos que publican por MQTT.
  • Plataformas como AWS IoT, Azure IoT Hub, ThingsBoard.
  • Sistemas SCADA o PLCs (exportando por OPC-UA, Modbus, HTTP).
  • Gateways industriales (convirtiendo datos a JSON/REST/MQTT).

Flujo de Datos Desacoplado:

[Dispositivo IoT] → [Backend / Broker MQTT] → [API] → [Dashboard]

El frontend puede recibir datos en tiempo real, actualizaciones push, o usar WebSockets/MQTT directamente.

Widgets para Visualización y Control

🔢 Sensores: Temperatura, humedad, voltaje, etc.
📊 Gráficas: Barras, donut, series de tiempo.
🚦 Alarmas y Estados: On/Off, fallas, niveles críticos.
🧱 Controles: Actuadores, sliders, botones.
🧩 Gemelos Digitales Visuales: Modelos 2D/3D, paneles.

Capacidades Clave de Visualización IoT en 3D

  • Sprites y Sensores 2D: Iconos en el modelo 3D que representan dispositivos IoT, mostrando datos detallados al interactuar.
  • Mapas de Calor (Heatmaps): Superposición de colores en el modelo para indicar valores (temperatura, CO₂, etc.), facilitando la detección visual de zonas críticas.
  • Datos en Tiempo Real: Actualización dinámica de métricas en el modelo, con herramientas de timeline para análisis históricos.
  • Conectividad Múltiple: Adaptadores para bases de datos IoT comunes (Azure Time Series, AWS IoT, etc.).
  • Gráficos y Dashboards Embebidos: Integración de librerías (ej. Apache ECharts) para mostrar indicadores y alertas en la interfaz 3D.

Ejemplos de Sensores IoT Visualizables

Cualquier sensor conectado a una plataforma IoT puede integrarse. Algunos ejemplos:

🌡️ Temperatura (°C/°F) 💧 Humedad Relativa (%) 💨 Presión (bar, Pa) 👋 Vibración (acelerómetros) 🏭 Gases/Partículas (CO₂, CO, CH₄) ⚡ Medidores de Energía (kW, kWh)

Ideal para Proyectos Como

🏭 Fábricas Inteligentes: Monitoreo de producción, calidad, mantenimiento predictivo.
🌱 Agricultura de Precisión: Monitoreo de cultivos, riego, condiciones climáticas.
🏗️ Monitoreo Estructural: Vigilancia de puentes, edificios, infraestructuras críticas.
⚡ Energía y Utilities: Gestión de redes eléctricas, subestaciones, optimización de consumo.
🏠 Domótica y Smart Cities: Control de iluminación, climatización, seguridad.

Casos de Uso por Industria

⛏️ Minería

Supervisión de equipos (temperatura motor, vibraciones, combustible) y entornos remotos (gases peligrosos, calidad del aire) para mejorar seguridad y eficiencia.

🏗️ Construcción

Vigilancia de integridad estructural (deformación, inclinación) y condiciones ambientales en obra (curado de hormigón, confort).

💡 Energía

Modelado de subestaciones, parques eólicos, plantas generadoras. Monitoreo remoto de transformadores, voltaje, corriente, temperatura para mantenimiento predictivo.

Beneficios Empresariales

  • Mayor eficiencia operativa: Decisiones proactivas con datos en tiempo real, optimización de rutinas.
  • Reducción de costos de mantenimiento: Mantenimiento preventivo basado en predicción de fallas.
  • Decisiones basadas en datos: Información objetiva y visual para la dirección.
  • Mejora en seguridad y sustentabilidad: Advertencia de riesgos, protección de personal y medio ambiente.
